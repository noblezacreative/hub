# Project Hub v1

## Goal
Create a simple browser-based project hub to manage AI-driven projects.

## Core Flow
1. User creates a project
2. User uploads or attaches a PRD
3. User sees project dashboard
4. User can trigger an AI task manually
5. System logs output

## Features
- Project list
- Project detail page
- PRD attachment
- Task runner button
- Output log

## Constraints
- Keep it simple
- Browser-based
- No automation yet
- Focus on clarity

## Future Vision
This will evolve into a system with:
- memory
- multi-agent orchestration
- self-healing workflows
